# iot-lab-projects
# 🌐 IoT Project Collection
Kumpulan proyek IoT menggunakan Arduino, ESP, dan sensor, mencakup robotika, otomasi, dan sistem pintar. Collection of IoT projects using Arduino, ESP, and sensors including robotics, automation, and smart systems.

## 🚀 Tentang Repository Ini
Repository ini berisi kumpulan eksperimen dan proyek Internet of Things (IoT) yang saya buat untuk pembelajaran dan pengembangan sistem embedded, seperti:

- 🤖 Robotika dan sistem embedded
- 📡 Sensor dan sistem deteksi (ultrasonic, dll)
- 🔌 Sistem otomasi sederhana
- 🧠 Logika kontrol berbasis Arduino/ESP

---

## 🛠️ Teknologi yang Digunakan
- Arduino Uno / Nano
- ESP8266 / ESP32 (opsional di beberapa project)
- Servo Motor
- Sensor 
- LED & komponen elektronik dasar
- Bahasa pemrograman C/C++ (Arduino IDE)

---

## 🎯 Tujuan Project
- Belajar dasar IoT dan embedded system
- Mengembangkan logika robotika sederhana
- Eksperimen sensor dan aktuator
- Membangun portfolio project engineering

## 📌 Catatan
